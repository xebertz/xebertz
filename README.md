- **TASkOcupado** — App en Java para **crear/asignar tareas** (MVC + Observer) con repos separados (Core/UI/Config + notifiers por Email/Telegram + logger).

  - **[TelegramNotifier](https://github.com/pp2-ungs/TelegramNotifier)** — Notificador por Telegram para las asignaciones.
  - **[AssignmentLogger](https://github.com/pp2-ungs/AssignmentLogger)** — Logger externo para registrar el historial de asignaciones.

- **[PenTesAgent](https://github.com/xebertz/PenTestAgent)** — Pipeline de **ML** para clasificar vulnerabilidades desde **tráfico de red** (dataset con `tshark` + CICFlowMeter; modelos RF y NN con TensorFlow/Keras) + **agente autónomo** que realiza discovery/probing y **movimiento horizontal/lateral** basado en la predicción del modelo.

- **[BIOS](https://github.com/xebertz/BIOS)** — Proyecto en **Python** de **clasificación de imágenes médicas** para **detectar enfermedades mediante IA**, con **3 modelos de IA** para distintos diagnósticos.
